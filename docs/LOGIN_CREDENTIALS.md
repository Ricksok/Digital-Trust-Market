## Trust Scores

After seeding, all users have trust scores calculated based on:
- KYC status (Identity Trust)
- Investment/payment activity (Transaction & Financial Trust)
- Project success (Performance Trust)
- Course completion (Learning Trust)

**Typical Trust Score Ranges:**
- **High Trust (80-100):** Verified users with successful activity
- **Medium Trust (50-79):** Users with some activity but not verified
- **Low Trust (0-49):** New users or users with failed transactions

## Resetting Data

To reset all data and recreate users:

```bash
cd backend
npm run db:seed
```

Or in Docker:

```bash
docker-compose exec backend npm run db:seed
```

This will:
- Clear all existing data
- Create fresh users with same credentials
- Generate new projects and investments
- Calculate new trust scores
